Cloudflare Bypass

A lightweight, self-hosted API server that bypasses Cloudflare Turnstile protection. No third-party dependencies. Built entirely with Playwright.

View on GitHub Quick Start

Why Cloudflare Bypass?

Zero Third-Party Dependencies

No FlareSolverr, no puppeteer-extra, no cloudscraper. 100% own code with Playwright.

Anti-Detection Built In

Removes navigator.webdriver, spoofs plugins, matches locale/timezone. Your requests look human.

Cookie Reuse

Get cf_clearance once, reuse it across curl, Python, JS, or Playwright. Fast and efficient.

Simple API

Single POST endpoint. JSON in, JSON out. Integrates in minutes with any language.

Quick Start

# Install & run
git clone https://github.com/Radit-lab/cloudflare-bypass.git
cd cloudflare-bypass
npm install
npm start
# Bypass any Cloudflare-protected URL
curl -X POST http://localhost:8191/cloudflare \
  -H "Content-Type: application/json" \
  -d '{"mode":"iuam","domain":"https://example.com"}'
